Variable: comint-prompt-regexp

Regexp to recognize prompts in the inferior process. Defaults to "^", the null string at BOL.

This variable is only used if the variable `comint-use-prompt-regexp' is non-nil.

Good choices: Canonical Lisp: "^[^> \n]*>+:? *" (Lucid, franz, kcl, T, cscheme, oaklisp) Lucid Common Lisp: "^\\(>\\|\\(->\\)+\\) *" franz: "^\\(->\\|<[0-9]*>:\\) *" kcl: "^>+ *" shell: "^[^#$%>\n]*[#$%>] *" T: "^>+ *"

This is a good thing to set in mode hooks.
