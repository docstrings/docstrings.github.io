Function: list-dynamic-libraries

Display a list of all dynamic libraries known to Emacs. (These are the libraries listed in `dynamic-library-alist'.) If optional argument LOADED-ONLY-P (interactively, prefix arg) is non-nil, only libraries already loaded are listed. Optional argument BUFFER specifies a buffer to use, instead of "*Dynamic Libraries*". The return value is always nil.
