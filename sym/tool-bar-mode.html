Function: tool-bar-mode

Toggle the tool bar in all graphical frames (Tool Bar mode). With a prefix argument ARG, enable Tool Bar mode if ARG is positive, and disable it otherwise. If called from Lisp, enable Tool Bar mode if ARG is omitted or nil.

See `tool-bar-add-item' and `tool-bar-add-item-from-menu' for conveniently adding tool bar items.

Variable: tool-bar-mode

Non-nil if Tool-Bar mode is enabled. See the command `tool-bar-mode' for a description of this minor mode. Setting this variable directly does not take effect; either customize it (see the info node `Easy Customization') or call the function `tool-bar-mode'.
