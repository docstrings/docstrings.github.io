Function: ediff-regions-linewise

Run Ediff on a pair of regions in specified buffers. Regions (i.e., point and mark) can be set in advance or marked interactively. Each region is enlarged to contain full lines. This function is effective for large regions, over 100-200 lines. For small regions, use `ediff-regions-wordwise'.
