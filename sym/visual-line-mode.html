Function: visual-line-mode

Toggle visual line based editing (Visual Line mode). With a prefix argument ARG, enable Visual Line mode if ARG is positive, and disable it otherwise. If called from Lisp, enable the mode if ARG is omitted or nil.

When Visual Line mode is enabled, `word-wrap' is turned on in this buffer, and simple editing commands are redefined to act on visual lines, not logical lines. See Info node `Visual Line Mode' for details.

Variable: visual-line-mode

Non-nil if Visual-Line mode is enabled. Use the command `visual-line-mode' to change this variable.
