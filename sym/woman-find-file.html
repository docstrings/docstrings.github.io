Function: woman-find-file

Find, decode and browse a specific UN*X man-page source file FILE-NAME. Use existing buffer if possible; reformat only if prefix arg given. When called interactively, optional argument REFORMAT forces reformatting of an existing WoMan buffer formatted earlier. No external programs are used, except that `gunzip' will be used to decompress the file if appropriate. See the documentation for the `woman' command for further details.

(fn FILE-NAME &optional REFORMAT)
