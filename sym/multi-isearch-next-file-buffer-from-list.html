Function: multi-isearch-next-file-buffer-from-list

Return the next buffer in the series of file buffers. This function is used for multiple file buffers Isearch. A sequence of files is defined by the variable `multi-isearch-file-list' set in `multi-isearch-files' or `multi-isearch-files-regexp'. Every next/previous file in the defined sequence is visited by `find-file-noselect' that returns the corresponding file buffer.
