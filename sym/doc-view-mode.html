Function: doc-view-mode

Major mode in DocView buffers.

DocView Mode is an Emacs document viewer. It displays PDF, PS and DVI files (as PNG images) in Emacs buffers.

You can use Uses keymap `doc-view-mode-map', which is not currently defined. M-x doc-view-toggle-display to toggle between displaying the document or editing it as text.

Uses keymap `doc-view-mode-map', which is not currently defined.
