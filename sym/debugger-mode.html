Function: debugger-mode

Mode for backtrace buffers, selected in debugger.

A line starts with `*' if exiting that frame will call the debugger. Type b or u to set or remove the `*'.

When in debugger due to frame being exited, use the r command to override the value being returned from that frame.

Use M-x debug-on-entry and M-x cancel-debug-on-entry to control which functions will enter the debugger when called.
