Function: tpu-edt-mode

Toggle TPU/edt emulation on or off. With a prefix argument ARG, enable the mode if ARG is positive, and disable it otherwise. If called from Lisp, enable the mode if ARG is omitted or nil.

(fn &optional ARG)

Variable: tpu-edt-mode

Non-nil if Tpu-Edt mode is enabled. See the command `tpu-edt-mode' for a description of this minor mode. Setting this variable directly does not take effect; either customize it (see the info node `Easy Customization') or call the function `tpu-edt-mode'.
