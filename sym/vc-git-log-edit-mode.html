Function: vc-git-log-edit-mode

Major mode for editing Git log messages. It is based on `log-edit-mode', and has Git-specific extensions.

In addition to any hooks its parent mode might have run, this mode runs the hook `vc-git-log-edit-mode-hook', as the final step during initialization.

key binding --- -------

C-c Prefix Command
