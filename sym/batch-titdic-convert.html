Function: batch-titdic-convert

Run `titdic-convert' on the files remaining on the command line. Use this from the command line, with `-batch'; it won't work in an interactive Emacs. For example, invoke "emacs -batch -f batch-titdic-convert XXX.tit" to generate Quail package file "xxx.el" from TIT dictionary file "XXX.tit". To get complete usage, invoke "emacs -batch -f batch-titdic-convert -h".
