Variable: dabbrev-case-distinction

Whether dabbrev treats expansions as the same if they differ in case.

A value of nil means treat them as different. A value of `case-replace' means distinguish them if `case-replace' is nil. Any other non-nil value means to treat them as the same.

This variable has an effect only when the value of `dabbrev-case-fold-search' specifies to ignore case.
