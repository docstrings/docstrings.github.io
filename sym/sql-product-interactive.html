Function: sql-product-interactive

Run PRODUCT interpreter as an inferior process.

If buffer `*SQL*' exists but no process is running, make a new process. If buffer exists and a process is running, just switch to buffer `*SQL*'.

To specify the SQL product, prefix the call with C-u. To set the buffer name as well, prefix the call to M-x sql-product-interactive with C-u C-u.

(Type C-h m in the SQL buffer for a list of commands.)

(fn &optional PRODUCT NEW-NAME)
