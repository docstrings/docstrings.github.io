Function: electric-layout-mode

Automatically insert newlines around some chars. With a prefix argument ARG, enable Electric Layout mode if ARG is positive, and disable it otherwise. If called from Lisp, enable the mode if ARG is omitted or nil. The variable `electric-layout-rules' says when and how to insert newlines.

(fn &optional ARG)

Variable: electric-layout-mode

Non-nil if Electric-Layout mode is enabled. See the command `electric-layout-mode' for a description of this minor mode. Setting this variable directly does not take effect; either customize it (see the info node `Easy Customization') or call the function `electric-layout-mode'.
