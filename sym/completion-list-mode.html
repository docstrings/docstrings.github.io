Function: completion-list-mode

Major mode for buffers showing lists of possible completions. Type RET in the completion list to select the completion near point. Use to select one with the mouse.

key binding --- -------

RET choose-completion ESC Prefix Command q quit-window z kill-this-buffer mouse-face previous-completion mouse-choose-completion next-completion

ESC ESC Prefix Command

M-ESC ESC delete-completion-window

This mode runs the hook `completion-list-mode-hook', as the final step during initialization.
