Function: emacs-lisp-mode

Major mode for editing Lisp code to run in Emacs. Commands: Delete converts tabs to spaces as it moves back. Blank lines separate paragraphs. Semicolons start comments.

key binding --- -------

ESC Prefix Command DEL backward-delete-char-untabify

C-M-i completion-at-point C-M-q indent-pp-sexp C-M-x eval-defun

C-M-q indent-sexp (that binding is currently shadowed by another mode)

Entry to this mode calls the value of `emacs-lisp-mode-hook' if that value is non-nil.
