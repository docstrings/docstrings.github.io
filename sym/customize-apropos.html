Function: customize-apropos

Customize loaded options, faces and groups matching PATTERN. PATTERN can be a word, a list of words (separated by spaces), or a regexp (using some regexp special characters). If it is a word, search for matches for that word as a substring. If it is a list of words, search for matches for any two (or more) of those words.

If TYPE is `options', include only options. If TYPE is `faces', include only faces. If TYPE is `groups', include only groups.

(fn PATTERN &optional TYPE)
