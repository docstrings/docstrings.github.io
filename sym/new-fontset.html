Function: new-fontset

Create a new fontset NAME from font information in FONTLIST.

FONTLIST is an alist of scripts vs the corresponding font specification list. Each element of FONTLIST has the form (SCRIPT FONT-SPEC ...), where a character of SCRIPT is displayed by a font that matches one of FONT-SPEC.

SCRIPT is a symbol that appears in the first extra slot of the char-table `char-script-table'.

FONT-SPEC is a vector, a cons, or a string. See the documentation of `set-fontset-font' for the meaning.

(fn NAME FONTLIST)
