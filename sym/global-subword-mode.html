Function: global-subword-mode

Toggle Subword mode in all buffers. With prefix ARG, enable Global-Subword mode if ARG is positive; otherwise, disable it. If called from Lisp, enable the mode if ARG is omitted or nil.

Subword mode is enabled in all buffers where `(lambda nil (subword-mode 1))' would do it. See `subword-mode' for more information on Subword mode.

(fn &optional ARG)

Variable: global-subword-mode

Non-nil if Global-Subword mode is enabled. See the command `global-subword-mode' for a description of this minor mode. Setting this variable directly does not take effect; either customize it (see the info node `Easy Customization') or call the function `global-subword-mode'.
