Function: Info-file-supports-index-cookies

Return non-nil value if FILE supports Info index cookies. Info index cookies were first introduced in 4.7, and all later makeinfo versions output them in index nodes, so we can rely solely on the makeinfo version. This function caches the information in `Info-file-supports-index-cookies-list'.

Variable: Info-file-supports-index-cookies

Non-nil if current Info file supports index cookies.
