Function: find-file-other-frame

Edit file FILENAME, in another frame.

Like C-x C-f (which see), but creates a new frame or reuses an existing one. See the function `display-buffer'.

Interactively, the default if you just type RET is the current directory, but the visited file name is available through the minibuffer history: type M-n to pull it into the minibuffer.

Interactively, or if WILDCARDS is non-nil in a call from Lisp, expand wildcards (if any) and visit multiple files.
