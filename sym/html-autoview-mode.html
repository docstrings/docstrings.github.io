Function: html-autoview-mode

Toggle viewing of HTML files on save (HTML Autoview mode). With a prefix argument ARG, enable HTML Autoview mode if ARG is positive, and disable it otherwise. If called from Lisp, enable the mode if ARG is omitted or nil.

HTML Autoview mode is a buffer-local minor mode for use with `html-mode'. If enabled, saving the file automatically runs `browse-url-of-buffer' to view it.

Variable: html-autoview-mode

Non-nil if Html-Autoview mode is enabled. Use the command `html-autoview-mode' to change this variable.
