Function: url-handler-mode

Toggle using `url' library for URL filenames (URL Handler mode). With a prefix argument ARG, enable URL Handler mode if ARG is positive, and disable it otherwise. If called from Lisp, enable the mode if ARG is omitted or nil.

(fn &optional ARG)

Variable: url-handler-mode

Non-nil if Url-Handler mode is enabled. See the command `url-handler-mode' for a description of this minor mode. Setting this variable directly does not take effect; either customize it (see the info node `Easy Customization') or call the function `url-handler-mode'.
