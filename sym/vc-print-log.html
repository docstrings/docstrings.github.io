Function: vc-print-log

List the change log of the current fileset in a window. If WORKING-REVISION is non-nil, leave point at that revision. If LIMIT is non-nil, it should be a number specifying the maximum number of revisions to show; the default is `vc-log-show-limit'.

When called interactively with a prefix argument, prompt for WORKING-REVISION and LIMIT.

(fn &optional WORKING-REVISION LIMIT)
