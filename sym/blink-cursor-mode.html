Function: blink-cursor-mode

Toggle cursor blinking (Blink Cursor mode). With a prefix argument ARG, enable Blink Cursor mode if ARG is positive, and disable it otherwise. If called from Lisp, enable the mode if ARG is omitted or nil.

This command is effective only on graphical frames. On text-only terminals, cursor blinking is controlled by the terminal.

Variable: blink-cursor-mode

Non-nil if Blink-Cursor mode is enabled. See the command `blink-cursor-mode' for a description of this minor mode. Setting this variable directly does not take effect; either customize it (see the info node `Easy Customization') or call the function `blink-cursor-mode'.
