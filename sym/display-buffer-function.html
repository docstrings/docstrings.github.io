Variable: display-buffer-function

If non-nil, function to call to handle `display-buffer'. It will receive two args, the buffer and a flag which if non-nil means that the currently selected window is not acceptable. It should choose or create a window, display the specified buffer in it, and return the window.

The specified function should call `display-buffer-record-window' with corresponding arguments to set up the quit-restore parameter of the window used.
