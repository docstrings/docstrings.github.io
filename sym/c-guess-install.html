Function: c-guess-install

Install the latest guessed style into the current buffer. (This guessed style is a combination of `c-guess-guessed-basic-offset', `c-guess-guessed-offsets-alist' and `c-offsets-alist'.)

The style is entered into CC Mode's style system by `c-add-style'. Its name is either STYLE-NAME, or a name based on the absolute file name of the file if STYLE-NAME is nil.

(fn &optional STYLE-NAME)
