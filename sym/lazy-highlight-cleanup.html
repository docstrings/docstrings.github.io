Function: lazy-highlight-cleanup

Stop lazy highlighting and remove extra highlighting from current buffer. FORCE non-nil means do it whether or not `lazy-highlight-cleanup' is nil. This function is called when exiting an incremental search if `lazy-highlight-cleanup' is non-nil.

Variable: lazy-highlight-cleanup

Controls whether to remove extra highlighting after a search. If this is nil, extra highlighting can be "manually" removed with M-x lazy-highlight-cleanup.
