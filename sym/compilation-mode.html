Function: compilation-mode

Major mode for compilation log buffers.

Uses keymap `compilation-mode-map', which is not currently defined. To visit the source for a line-numbered error, move point to the error message line and type M-x compile-goto-error. To kill the compilation, type M-x kill-compilation.

Runs `compilation-mode-hook' with `run-mode-hooks' (which see).

Uses keymap `compilation-mode-map', which is not currently defined.
