Function: safe-local-variable-p

Non-nil if SYM is safe as a file-local variable with value VAL. It is safe if any of these conditions are met:

* There is a matching entry (SYM . VAL) in the `safe-local-variable-values' user option.

* The `safe-local-variable' property of SYM is a function that evaluates to a non-nil value with VAL as an argument.
