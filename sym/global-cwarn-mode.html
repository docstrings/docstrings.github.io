Function: global-cwarn-mode

Toggle Cwarn mode in all buffers. With prefix ARG, enable Global-Cwarn mode if ARG is positive; otherwise, disable it. If called from Lisp, enable the mode if ARG is omitted or nil.

Cwarn mode is enabled in all buffers where `turn-on-cwarn-mode-if-enabled' would do it. See `cwarn-mode' for more information on Cwarn mode.

(fn &optional ARG)

Variable: global-cwarn-mode

Non-nil if Global-Cwarn mode is enabled. See the command `global-cwarn-mode' for a description of this minor mode. Setting this variable directly does not take effect; either customize it (see the info node `Easy Customization') or call the function `global-cwarn-mode'.
