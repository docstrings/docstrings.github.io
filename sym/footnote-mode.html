Function: footnote-mode

Toggle Footnote mode. With a prefix argument ARG, enable Footnote mode if ARG is positive, and disable it otherwise. If called from Lisp, enable the mode if ARG is omitted or nil.

Uses keymap `footnote-minor-mode-map', which is not currently defined.
