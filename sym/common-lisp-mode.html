Function: common-lisp-mode

Major mode for editing Lisp code for Lisps other than GNU Emacs Lisp. Commands: Delete converts tabs to spaces as it moves back. Blank lines separate paragraphs. Semicolons start comments.

key binding --- -------

C-c Prefix Command ESC Prefix Command DEL backward-delete-char-untabify

C-c C-z run-lisp

C-M-x lisp-eval-defun

C-M-q indent-sexp

Note that `run-lisp' may be used either to start an inferior Lisp job or to switch back to an existing one.

Entry to this mode calls the value of `lisp-mode-hook' if that value is non-nil.
