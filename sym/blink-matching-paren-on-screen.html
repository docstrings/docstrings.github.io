Variable: blink-matching-paren-on-screen

Non-nil means show matching open-paren when it is on screen. If nil, don't show it (but the open-paren can still be shown when it is off screen).

This variable has no effect if `blink-matching-paren' is nil. (In that case, the open-paren is never shown.) It is also ignored if `show-paren-mode' is enabled.
