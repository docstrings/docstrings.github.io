Function: tooltip-mode

Toggle Tooltip mode. With a prefix argument ARG, enable Tooltip mode if ARG is positive, and disable it otherwise. If called from Lisp, enable the mode if ARG is omitted or nil.

When this global minor mode is enabled, Emacs displays help text (e.g. for buttons and menu items that you put the mouse on) in a pop-up window.

When Tooltip mode is disabled, Emacs displays help text in the echo area, instead of making a pop-up window.

Variable: tooltip-mode

Non-nil if Tooltip mode is enabled. See the command `tooltip-mode' for a description of this minor mode. Setting this variable directly does not take effect; either customize it (see the info node `Easy Customization') or call the function `tooltip-mode'.
