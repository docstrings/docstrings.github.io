Function: autoload-do-load

Load FUNDEF which should be an autoload. If non-nil, FUNNAME should be the symbol whose function value is FUNDEF, in which case the function returns the new autoloaded function value. If equal to `macro', MACRO-ONLY specifies that FUNDEF should only be loaded if it is defines a macro.

(fn FUNDEF &optional FUNNAME MACRO-ONLY)
