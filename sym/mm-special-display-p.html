Function: mm-special-display-p

Return non-nil if a buffer named BUFFER-NAME gets a special frame. More precisely, return t if `special-display-buffer-names' or `special-display-regexps' contain a string entry equaling or matching BUFFER-NAME. If `special-display-buffer-names' or `special-display-regexps' contain a list entry whose car equals or matches BUFFER-NAME, the return value is the cdr of that entry.
