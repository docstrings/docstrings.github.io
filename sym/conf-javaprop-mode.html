Function: conf-javaprop-mode

Conf Mode starter for Java properties files. Comments start with `#' but are also recognized with `//' or between `/*' and `*/'. For details see `conf-mode'. Example:

# Conf mode font-locks this right with M-x conf-javaprop-mode (Java properties) // another kind of comment /* yet another */

name:value name=value name value x.1 = x.2.y.1.z.1 = x.2.y.1.z.2.zz =
