Variable: allout-widgets-auto-activation

Activate to enable allout icon graphics wherever allout mode is active.

Also enable `allout-auto-activation' for this to take effect upon visiting an outline.

When this is set you can disable allout widgets in select files by setting `allout-widgets-mode-inhibit'

Instead of setting `allout-widgets-auto-activation' you can explicitly invoke `allout-widgets-mode' in allout buffers where you want allout widgets operation.

See `allout-widgets-mode' for allout widgets mode features.
