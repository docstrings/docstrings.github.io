Function: org-store-link

Uses keymap `org-mode-map', which is not currently defined. Store an org-link to the current location. This link is added to `org-stored-links' and can later be inserted into an org-buffer with M-x org-insert-link.

For some link types, a prefix arg is interpreted: For links to usenet articles, arg negates `org-gnus-prefer-web-links'. For file links, arg negates `org-context-in-file-links'.
