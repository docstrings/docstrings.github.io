Variable: font-lock-removed-keywords-alist

Alist of `font-lock-keywords' elements to be removed for major modes.

Each element has the form (MODE . KEYWORDS). `font-lock-set-defaults' removes the elements in the list KEYWORDS from `font-lock-keywords' when Font Lock is turned on in major mode MODE.

This is normally set via `font-lock-add-keywords' and `font-lock-remove-keywords'.
