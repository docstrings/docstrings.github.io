Variable: mm-codepage-ibm-list

List of IBM codepage numbers.

The codepage mappings slightly differ between IBM and other vendors. See "ftp://ftp.unicode.org/Public/MAPPINGS/VENDORS/IBM/README.TXT".

If an element is a number corresponding to a supported windows codepage, appropriate entries to `mm-charset-synonym-alist' are added by `mm-setup-codepage-ibm'.
