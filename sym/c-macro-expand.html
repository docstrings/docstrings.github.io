Function: c-macro-expand

Expand C macros in the region, using the C preprocessor. Normally display output in temp buffer, but prefix arg means replace the region with it.

`c-macro-preprocessor' specifies the preprocessor to use. Tf the user option `c-macro-prompt-flag' is non-nil prompt for arguments to the preprocessor (e.g. `-DDEBUG -I ./include'), otherwise use `c-macro-cppflags'.

Noninteractive args are START, END, SUBST. For use inside Lisp programs, see also `c-macro-expansion'.
