Function: set-face-stipple

Change the stipple pixmap of face FACE to STIPPLE. FRAME nil or not specified means change face on all frames. STIPPLE should be a string, the name of a file of pixmap data. The directories listed in the `x-bitmap-file-path' variable are searched.

Alternatively, STIPPLE may be a list of the form (WIDTH HEIGHT DATA) where WIDTH and HEIGHT are the size in pixels, and DATA is a string, containing the raw bits of the bitmap.
