Variable: comint-move-point-for-output

Controls whether interpreter output moves point to the end of the output. If nil, then output never moves point to the output. (If the output occurs at point, it is inserted before point.) If t or `all', move point in all windows showing the buffer. If `this', move point only the selected window. If `others', move point only in other windows, not in the selected window.

The default is nil.

See the variable `comint-scroll-show-maximum-output' and the function `comint-postoutput-scroll-to-bottom'. This variable is buffer-local in all Comint buffers.
