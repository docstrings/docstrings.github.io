Function: gnus-select-frame-set-input-focus

Select FRAME, raise it, and set input focus, if possible. If `mouse-autoselect-window' is non-nil, also move mouse pointer to FRAME's selected window. Otherwise, if `focus-follows-mouse' is non-nil, move mouse cursor to FRAME.

Optional argument NORECORD means to neither change the order of recently selected windows nor the buffer list.
