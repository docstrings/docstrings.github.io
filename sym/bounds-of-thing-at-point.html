Function: bounds-of-thing-at-point

Determine the start and end buffer locations for the THING at point. THING should be a symbol specifying a type of syntactic entity. Possibilities include `symbol', `list', `sexp', `defun', `filename', `url', `email', `word', `sentence', `whitespace', `line', and `page'.

See the file `thingatpt.el' for documentation on how to define a valid THING.

Return a cons cell (START . END) giving the start and end positions of the thing found.

(fn THING)
